## Quick Fixes

**Extension not working?**
- Wait for Dart Analysis Server indexing
- Increase `dartUnusedCode.analysisDelay` to 3000-5000ms
- Check logs: View → Output → Dart Unused Code

**False positives?**
- Wait for indexing to complete
- Re-run: Cmd+Shift+P → "Dart: Analyze Unused Code"
- Adjust `dartUnusedCode.excludePatterns`
